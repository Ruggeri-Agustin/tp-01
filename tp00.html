br: produce un salto de línea en el texto (retorno de carro). Es útil para escribir un poema o una dirección, donde la división de las líneas es significante, u: Muestra el texto subrayado, sub:define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice, sup: define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice, em:es el apropiado para marcar con énfasis las partes importantes de un texto. El elemento "em" puede ser anidado, con cada nivel de anidamiento indicando un mayor grado de énfasis, strong:El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto, i: Muestra el texto marcado con un estilo en cursiva o italica, p: sirve para escribir texto

12-especificar el uso de las siguentes etiquetas: <blockquote></blockquote>, <cite></cite>, <hr>, <abbr></abbr>, <code><code>, <span></span>, <a></a>

-"blockquote":Crea citas en bloque, marca las citas a otros autores o documentos.ej:

hola mundo

-"cite":Marca una referencia a una fuente, o el autor de un texto citado.ej: hola mundo

-"hr":representa un cambio de tema entre párrafos (por ejemplo, un cambio de escena en una historia, un cambio de tema en una sección).ej:

hola mundo

-"abbr":representa una abreviación o acrónimo; el atributo opcional title puede ampliar o describir la abreviatura. ej: hola mundo

-"code":Es el apropiado para marcar el código de un programa.ej: hola mundo

-"span":Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea. ej: hola mundo

-"a":crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL. ej: hola mundo
